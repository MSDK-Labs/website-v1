How it works

From decision to certificate.

AI System

Your model produces a decision with inputs, outputs, and metadata.

Aegis Trace API

Decision payload submitted via single REST endpoint.

PII Processing

Personal data fields tokenised per configurable schema rules.

Cryptographic Signing

HMAC-SHA256 certificate generated with managed key infrastructure.

Immutable Storage

Certificate written to WORM-locked append-only store.
